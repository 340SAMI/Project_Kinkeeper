# 🏠 Project Kinkeeper

> *Coordinating care, connecting families — one task at a time.*

---

## 📖 About

**Project Kinkeeper** is a  coordination app that helps families and caregivers stay organized, informed, and in sync. Whether you're managing care for an elderly parent, a child, or anyone who needs support, Kinkeeper brings everyone on the care team together — tracking tasks, schedules, and responsibilities so no one carries the burden alone.

## ✨ Key Features

### 1. 📝 Simple Task Creation
Users can easily add caregiving or daily tasks through an input field, making it quick to track responsibilities.

### 2. 📋 Task List Display
All added tasks are displayed in a structured list format, allowing users to clearly see what needs to be done.

### 3. ❌ Task Removal
Users can remove completed or unnecessary tasks instantly, keeping the list clean and up to date.

### 4. 💾 Local Storage Persistence
Tasks are saved in the browser using localStorage, so data remains even after refreshing the page.

### 5. 📱 Responsive UI
The interface adapts to different screen sizes, ensuring usability across desktop and mobile devices.

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/340SAMI/Project_Kinkeeper.git

# Navigate to the project folder
cd Project_Kinkeeper

# Install dependencies
npm install

# Start the development server
npm start
```
